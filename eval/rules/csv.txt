CSV TASK (CRITICAL):

- You MUST import csv and glob
- You MUST search recursively for CSV files
- Prefer files under data/
- Inspect headers first with csv.DictReader
- Read ALL rows:
    rows = list(reader)

LIST/DICT SAFETY (CRITICAL):
- rows is a LIST after rows = list(reader)
- NEVER call rows.keys()
- Use reader.fieldnames or the first row dict for schema inspection:
    headers = reader.fieldnames or []
    first_row = rows[0] if rows else {{}}
- When using dynamic key detection, apply it to a row dict, not to the rows list

FIELD DETECTION:
- Detect category-like fields dynamically:
    category, type, group, class
- Detect amount-like fields dynamically:
    total, revenue, amount, sales, price, unit_price
- Detect quantity-like fields dynamically:
    quantity, qty, count, units
- Detect date-like fields dynamically:
    date, order_date, timestamp, created_at

REVENUE LOGIC:
- If total exists and is numeric -> use total directly as row revenue
- Else if quantity and unit_price exist -> revenue = quantity * unit_price
- Else skip only the invalid row
- NEVER discard the whole dataset because some rows are invalid

TASK-SPECIFIC INTERPRETATION:
- If task asks for "total revenue per category", compute totals for the FULL dataset grouped by category
- If task ALSO asks which category had the highest revenue in December 2024 specifically:
    1. compute full-dataset category totals
    2. separately filter rows to December 2024
    3. compute December category totals
    4. identify the December highest category from the December totals only
- Do NOT use full-period totals when answering a December-specific question

DATE FILTERING:
- Parse date values carefully
- Accept formats like:
    YYYY-MM-DD
    YYYY-MM-DD HH:MM:SS
    ISO timestamps
- For December 2024, include only rows where year=2024 and month=12

OUTPUT:
- Return exact numeric results rounded to 2 decimals
- If the task contains both overall totals and a December-specific maximum, return both

PREFERRED OUTPUT SHAPE FOR THIS TYPE OF TASK:
{{
  "category_totals": {{}},
  "december_2024_totals": {{}},
  "highest_category_december_2024": "...",
  "highest_revenue_december_2024": 0.0
}}

INVALID OUTPUTS:
- Returning "No matching file found" without recursive search is invalid
- Returning empty result on non-empty CSV is invalid
- Returning only full-period totals for a month-specific question is invalid
