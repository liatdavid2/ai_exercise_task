FILE DISCOVERY (CRITICAL):

- You MUST search using BOTH:

    glob.glob('data/**/*.ext', recursive=True)
    glob.glob('**/*.ext', recursive=True)

- Combine results:

    files = glob.glob('data/**/*.ext', recursive=True) + glob.glob('**/*.ext', recursive=True)

- Remove duplicates

- Prefer files inside 'data/' if exist

- NEVER assume only one location



NO FILE FOUND HANDLING (CRITICAL):

- If no file found:
    - DO NOT immediately return

    - Try fallback:
        Try os.walk('data/')

    - ONLY if still nothing:
        return "No matching file found"


JSON HANDLING (CRITICAL):

- If JSON file contains list:
    iterate over items

- If dict:
    use values()

- You MUST inspect structure:

    data = json.load(f)

    if isinstance(data, list):
        rows = data
    elif isinstance(data, dict):
        rows = list(data.values())


 SAFE VALUE PARSING (CRITICAL):

- Never convert raw values directly without validation
- Before numeric conversion:
    - read the raw value safely
    - convert to string
    - strip whitespace
    - skip empty values

Recommended pattern:
    raw_value = str(value).strip()
    if not raw_value:
        continue
    try:
        numeric_value = float(raw_value)
    except:
        continue

- A single malformed value must not fail the whole task


TIME FILTERING (CRITICAL):

- If the task mentions a month, year, specific period, or date range:
    - infer the date field from the schema
    - parse date values safely
    - filter rows to the requested period before computing period-specific metrics

- Keep full-data calculations separate from period-specific calculations when the task asks for both
