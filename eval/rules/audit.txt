AUDIT CHECKS (MANDATORY):

For CSV / tabular files:
- Detect missing or empty values
- Detect duplicate rows or duplicate business keys when obvious
- Detect invalid numeric values
- Detect negative quantities / returns if present
- Detect missing totals
- Detect inconsistent totals vs quantity * unit_price when possible
- Detect unexpected categorical values such as unknown currencies

For JSON files:
- Detect missing required-like fields based on dominant schema
- Detect empty/null values
- Detect duplicate IDs / keys when possible
- Detect inconsistent field types across records

For LOG files:
- Detect malformed lines
- Detect multi-line stack traces
- Detect inconsistent timestamp formats
- Detect unexpected or partial log entries

For DB files:
- Inspect tables and schema first
- Detect nulls / duplicates / invalid values where possible
- If aggregate/summary tables exist, recompute metrics from raw tables and compare
- Report any mismatch between stored aggregates and recomputed values
